Добро пожаловать в систему управления персоналом!

Это программное обеспечение создано для удобного и эффективного управления информацией о физических лицах и сотрудниках вашей компании. Система обеспечивает надежное хранение, обновление и анализ данных, предоставляя простой и интуитивно понятный интерфейс.

Основные возможности системы

  • Ведение данных о физических лицах (ФИО, дата рождения, адрес, телефон).
  • Управление данными сотрудников (должности, контакты и дополнительная информация).
  • Простой интерфейс для работы с данными.
  • Обработка информации о документах, образовании, зарплатах и должностях.

Учет физических лиц

Раздел "Физ Лица" позволяет управлять данными физических лиц.

Основные действия:

  • Просмотр данных: Таблица с полями: Фамилия, Имя, Отчество, Дата рождения, Адрес, Телефон.
  • Добавление: Нажмите кнопку "Добавить физ лицо", заполните обязательные поля и сохраните запись.
  • Редактирование: Для изменения записи используйте кнопку "Изменить".
  • Удаление: Для удаления записи нажмите кнопку "Удалить".

Совет: Используйте поиск для быстрого нахождения информации по фамилии или имени.

Учет сотрудников

Раздел "Сотрудники" предназначен для управления профессиональной информацией о сотрудниках.

Основные действия:

  • Просмотр данных: Таблица с основными полями: Номер записи, ID физического лица, Должность, Образование, Дата приема, Рейтинг и др.
  • Добавление: Нажмите "Добавить сотрудника", выберите физическое лицо и заполните дополнительные данные.
  • Редактирование: Для изменения данных используйте кнопку "Изменить".
  • Удаление: Удалите запись с помощью кнопки "Удалить". Обратите внимание, что это также удаляет связанные данные.

Примечание: Информация о сотруднике связана с данными физического лица. При изменении данных физического лица информация сотрудника обновляется автоматически.

Детальная информация о сотрудниках

В карточке каждого сотрудника содержатся его личные и профессиональные данные.

  • Личные данные: ФИО, дата рождения, адрес, телефон.
  • Образование: Уровень, учебное заведение, год окончания, специальность.
  • Профессиональные данные: Должность, зарплата, ранг, даты приема и увольнения.
  • Документы: Тип, серия, номер, дата выдачи, кем выдан.
  • Финансовая информация: Номер карты, банк.

Совет: Для более подробной информации откройте карточку сотрудника в разделе "Сотрудники".

С чего начать?

  • Добавьте данные о физических лицах в разделе "Физ Лица".
  • Перейдите в раздел "Сотрудники" и создайте записи для сотрудников, связав их с физическими лицами.
  • Используйте кнопки "Изменить" и "Удалить" для редактирования данных.
  • Проверьте правильность данных перед сохранением.

Рекомендации

  • Регулярно обновляйте данные, чтобы они оставались актуальными.
  • Перед сохранением убедитесь, что все обязательные поля заполнены.
  • Используйте раздел "Помощь" для решения возможных вопросов.
